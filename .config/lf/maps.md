# GUÍA DE ATAJOS

## Atajos default

### Navegación

| Tecla | Acción |
| :--- | :--- |
| `k` / `j` | Subir / Bajar |
| `h` / `l` | Atrás / Entrar |
| `gg` / `G` | Inicio / Final |
| `Ctrl` + `u` / `d` | Scroll rápido |

### Selección

| Tecla | Acción |
| :--- | :--- |
| `Espacio` | Seleccionar |
| `v` | Invertir selección |
| `u` | Quitar selección |

### Archivos / Portapapeles

| Tecla | Acción |
| :--- | :--- |
| `y` | Copiar |
| `d` | Cortar |
| `p` | Pegar |
| `c` | Limpiar portapapeles |

### Búsqueda

| Tecla | Acción |
| :--- | :--- |
| `/` | Buscar (Nativo) |
| `n` / `N` | Siguiente / Anterior |

### Sistema

| Tecla | Acción |
| :--- | :--- |
| `:` | Comando |
| `$` | Terminal |
| `q` | Salir |

## Atajos personalizados

### Navegación Inteligente

| Tecla | Acción |
| :--- | :--- |
| `f` | Buscar archivo (FZF) |
| `F` | Saltar a carpeta (FZF) |

### Operaciones

| Tecla | Acción |
| :--- | :--- |
| `e` | Abre nvim |
| `a` | Crear directorio/archivo |
| `x` | Comprimir archivos (Menú) |
| `D` | Borrar archivos |
| `w` | Drag & Drop inteligente |

### Terminales

| Tecla | Acción |
| :--- | :--- |
| `t` | Terminal Externa (Kitty) |
| `T` | Terminal Interna |

### Ayuda

| Tecla | Acción |
| :--- | :--- |
| `?` | Mostrar esta guía |
